# Glossary

## GTD Terms
| Term | Meaning |
|------|---------|
| **GTD** | Getting Things Done — David Allen's productivity methodology |
| **Next Action** | The very next physical action to move a task forward |
| **Someday/Maybe** | Captured ideas not committed to — reviewed periodically |
| **Waiting For** | Tasks blocked on someone else |
| **Context** | Where/how a task can be done (e.g. @home, @computer, @phone) |
| **Area** | Life area of focus (Work, Personal, Health, Finance, etc.) |

## App-Specific Terms
| Term | Meaning |
|------|---------|
| **STAT** | Highest priority level — drop everything |
| **EOD** | End of day — must complete today |
| **StatusPill** | UI component showing task/project status with color coding |
| **PriorityBadge** | UI component showing priority level badge |
| **ContextTag** | White pill with blue text, prefixed @, for context tags |
| **Inbox** | Default landing status for new tasks/projects — unprocessed |
